====================================================
PART 4: DEPLOY TO NETLIFY (10 minutes)
====================================================

STEP 1: Import Your Project
----------------------------
1. Go to: https://app.netlify.com
2. Click "Add new site"
3. Click "Import an existing project"
4. Click "Deploy with GitHub"
5. Find "demon-list-tracker" in the list
6. Click it


STEP 2: Add Environment Variables (IMPORTANT!)
-----------------------------------------------
Before clicking deploy, you need to add your Supabase keys!

1. Scroll down to "Environment variables"
2. Click "Add environment variables"
3. Add the first variable:
   - Key: VITE_SUPABASE_URL
   - Value: [paste your Project URL from Supabase]
   
4. Click "Add another variable"
5. Add the second variable:
   - Key: VITE_SUPABASE_ANON_KEY
   - Value: [paste your anon/public key from Supabase]


STEP 3: Configure Build Settings
---------------------------------
1. Build command: npm run build
2. Publish directory: dist
3. Click "Deploy [your-site-name]"


STEP 4: Wait for Deployment

====================================================
TROUBLESHOOTING
====================================================

Problem: "Supabase client error"
‚Üí Check that environment variables are correct
‚Üí Make sure you copied the FULL keys (no extra spaces)
‚Üí Redeploy: Site configuration ‚Üí Deploys ‚Üí Trigger deploy

Problem: Site won't load
‚Üí Check browser console (F12) for errors
‚Üí Make sure build completed successfully
‚Üí Check Netlify deploy log for errors

Problem: Data not saving
‚Üí Verify SQL script ran successfully in Supabase
‚Üí Check that tables exist: Supabase ‚Üí Table Editor
‚Üí Verify RLS policies are enabled

Problem: Build failed
‚Üí Check that ALL files uploaded to GitHub
‚Üí Verify build command is: npm run build
‚Üí Check deploy log in Netlify for specific error

Problem: Changes not appearing
‚Üí Hard refresh the page (Ctrl+Shift+R or Cmd+Shift+R)
‚Üí Wait 30 seconds - real-time sync can have delay
‚Üí Check Supabase Table Editor to see if data changed
